Православный календарь.

== Description ==

Православный календарь. Имена и именины со встроенным календарем. Жития святых, иконы, тропари и кондаки. Ежедневные Евангельские чтения. Праздники, посты, дни поминовения усопших..

== Installation ==

1. Загрузите и распакуйте zip-файл на свой компьютер.
2. Скопируйте (Upload/FTP) всю папку orthodoxcalendarru в папку плагинов /httpdocs/blog/wp-content/plugins на блоге.
3. Зайдите в административную панель блога, зайдите в консоль «Плагины», там увидите только что добавленный плагин "Православный календарь". Активируйте его.
4. Поместите виджет "Православный календарь" в боковую колонку.
